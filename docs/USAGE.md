# 使用指南

## 用户端使用

### 1. 注册账号

1. 访问平台首页
2. 点击「注册」标签
3. 输入用户名（至少 3 位）和密码（至少 6 位）
4. 点击「注册」按钮

### 2. 登录

1. 在「登录」标签下输入用户名和密码
2. 点击「登录」按钮
3. 登录成功后会自动跳转到主界面

### 3. 兑换积分

1. 点击「兑换积分」标签
2. 输入管理员提供的兑换码
3. 点击「兑换」按钮
4. 兑换成功后，积分会立即到账

### 4. 生成图片

1. 点击「生成图片」标签
2. 在 Prompt 输入框中输入你想要生成的图片描述（英文效果更好）
3. 选择宽高比（可选）
4. 选择输出格式（PNG 或 JPEG）
5. 输入 Seed（可选，用于复现相同结果）
6. 点击「生成图片」按钮
7. 等待生成完成（通常需要 10-30 秒）
8. 生成成功后会显示图片，可以右键保存

**注意**：
- 每次生成消耗 1 积分
- 积分不足时无法生成
- Prompt 建议使用英文，描述越详细效果越好

### 5. 查看生成记录

1. 点击「生成记录」标签
2. 点击「刷新记录」按钮
3. 查看历史生成的所有图片

## 管理端使用

### 1. 访问管理后台

访问 `https://your-worker-url.workers.dev/admin`

### 2. 登录

输入管理员密码（在 `wrangler.toml` 中配置的 `ADMIN_PASSWORD`）

### 3. 查看统计概览

在「统计概览」标签下可以看到：
- 总用户数
- 总生成数
- 兑换码总数
- 已使用兑换码数量
- 剩余总积分

### 4. 生成兑换码

1. 点击「生成兑换码」标签
2. 输入积分数量（每个兑换码的积分值）
3. 输入生成数量（一次生成多少个兑换码）
4. 点击「生成」按钮
5. 复制生成的兑换码分发给用户

### 5. 查看用户列表

1. 点击「用户列表」标签
2. 点击「刷新」按钮
3. 查看所有用户的信息：
   - 用户名
   - 剩余积分
   - 已生成图片数量
   - 注册时间

### 6. 查看生成记录

1. 点击「生成记录」标签
2. 点击「刷新」按钮
3. 查看所有用户的生成记录

## Prompt 编写技巧

### 基础结构

```
[主体] + [风格] + [细节] + [质量词]
```

### 示例

**简单示例**：
```
A cat sitting on a windowsill, watercolor style, soft lighting
```

**详细示例**：
```
A majestic lion standing on a cliff at sunset, cinematic lighting,
dramatic clouds, golden hour, highly detailed, 8k resolution,
photorealistic
```

### 常用风格词

- `photorealistic` - 照片级真实
- `cinematic` - 电影感
- `watercolor` - 水彩画
- `oil painting` - 油画
- `anime style` - 动漫风格
- `minimalist` - 极简主义
- `cyberpunk` - 赛博朋克

### 常用质量词

- `highly detailed` - 高度细节
- `8k resolution` - 8K 分辨率
- `professional photography` - 专业摄影
- `award winning` - 获奖作品
- `masterpiece` - 杰作

### 光照词

- `golden hour` - 黄金时刻
- `soft lighting` - 柔和光照
- `dramatic lighting` - 戏剧性光照
- `studio lighting` - 影棚光照
- `natural light` - 自然光

## 常见问题

### Q: 生成失败怎么办？

A: 可能的原因：
1. 积分不足 - 请先兑换积分
2. Prompt 包含敏感内容 - 修改 Prompt 后重试
3. 网络超时 - 稍后重试

### Q: 如何复现相同的图片？

A: 使用相同的 Prompt、宽高比和 Seed 值即可复现。

### Q: 生成需要多长时间？

A: 通常需要 10-30 秒，具体取决于服务器负载。

### Q: 可以生成什么尺寸的图片？

A: 支持多种宽高比：
- 1:1 (正方形)
- 4:3 / 3:4 (标准)
- 16:9 / 9:16 (宽屏/手机)
- 21:9 / 9:21 (超宽屏)

### Q: 兑换码可以重复使用吗？

A: 不可以，每个兑换码只能使用一次。

### Q: 忘记密码怎么办？

A: 目前不支持密码找回，请联系管理员重置。
